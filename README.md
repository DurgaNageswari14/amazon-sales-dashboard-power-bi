# Amazon Sales Dashboard - Power BI

## Project Overview
This project is an interactive Power BI dashboard developed using Amazon sales data. It is designed to analyze sales performance, order trends, product categories, payment methods, order status, brand performance, and city-wise sales distribution. The dashboard provides both a summary view of overall sales performance and a detailed transaction-level analysis.

## Objectives
- To analyze Amazon sales performance using key business metrics
- To monitor total sales, orders, average order value, quantity sold, and shipping cost
- To compare category-wise and brand-wise sales trends
- To study sales distribution by payment method and order status
- To identify city-level sales patterns and detailed order information

## Dashboard Pages

### Page 1: Amazon Sales Overview Dashboard
This page provides a high-level summary of Amazon sales performance using KPI cards and charts.

#### Key Performance Indicators (KPIs)
- **Total Sales:** 91.83M
- **Total Orders:** 100K
- **Average Order Value:** 918.26
- **Total Quantity Sold:** 300K
- **Total Shipping Cost:** 740.67K

#### Visualizations Included
- **Sales by Year and Month** – shows monthly sales trend over multiple years
- **Sales by Payment Method** – donut chart showing contribution of different payment modes such as Credit Card, Debit Card, UPI, Amazon Pay, and Net Banking
- **Sales by Brand** – compares brand-wise sales performance
- **Sales by Order Status** – displays sales distribution across Delivered, Shipped, Pending, Cancelled, and Returned orders
- **Sales by City and Country** – highlights location-wise sales distribution
- **Orders by Category** – shows category-wise order count
- **Category Slicer** – allows dashboard filtering by product category

### Page 2: Amazon Sales Details Dashboard
This page presents transaction-level sales details in tabular format for detailed analysis.

#### Fields Included
- Year
- Quarter
- Month
- Day
- Product Name
- Category
- Brand
- Total Quantity Sold
- Payment Method
- Total Sales
- Order Status
- City

#### Filters/Slicers Included
- Category

## Insights Generated
- The dashboard helps track total Amazon sales and order performance over time

- Payment method analysis shows customer payment preferences
- Order status analysis highlights completed, pending, cancelled, and returned orders
- Brand-wise and category-wise insights help identify top-performing products and segments
- City-level sales distribution supports geographical performance analysis
- The details page helps in examining individual transactions for deeper business understanding
