# Power BI Data Analytics Project

## 📊 Project Overview

This project is a comprehensive Power BI report developed to demonstrate different data analysis, visualization, and interactive reporting techniques.

The project covers Power BI platform concepts, DAX calculations, data modeling, waterfall analysis, hierarchy analysis, bookmarks, drill-through functionality, conditional formatting, and financial performance analysis.

The report consists of multiple interactive pages designed to present business data in a clear and meaningful way.

---

## 🎯 Project Objectives

- To understand the Power BI platform and its major components
- To perform sales analysis using DAX calculations
- To analyze worldwide box office performance
- To study category-wise sales contribution using a waterfall chart
- To perform location-based hierarchy analysis
- To analyze sales trends using time hierarchy
- To implement bookmarks for interactive report navigation
- To use drill-through for detailed regional analysis
- To apply conditional formatting for easy performance comparison
- To develop an interactive financial performance dashboard

---

## 📑 Dashboard Pages

### 1. Power BI Platform

Provides an overview of Power BI, its major components, data flow, and key benefits.

The page explains:

- Power BI Desktop
- Power BI Service
- Power BI Mobile
- Power BI Gateway
- Power BI Report Server
- End-to-End Data Flow
- Benefits of Power BI

---

### 2. Sales Comparison Using DAX

Analyzes monthly sales performance using DAX measures.

Key metrics include:

- This Year Sales
- Last Year Sales
- Growth Percentage
- Monthly Sales Comparison
- Sales Growth Trend

---

### 3. Worldwide Box Office Analysis

Provides insights into worldwide movie box office performance.

Key metrics include:

- Total Worldwide Gross
- Total Domestic Gross
- Total International Gross
- Total Movies
- Worldwide Gross by Year
- Top Movies by Worldwide Gross

---

### 4. Category-wise Sales Waterfall Analysis

Uses a waterfall chart to analyze yearly sales contribution across different product categories.

The report highlights increases, decreases, and total sales contribution.

---

### 5. Location Hierarchy Analysis

Analyzes sales performance based on geographical hierarchy.

The report supports analysis across:

- Region
- Postal Code
- State

This page helps identify sales performance across different locations.

---

### 6. Sales Trend Analysis by Time Hierarchy

Analyzes sales performance using a time-based hierarchy.

Interactive bookmark buttons allow users to view data by:

- Year
- Quarter
- Month

The report compares Sales, COGS, and Profit across different time periods.

---

### 7. Regional Profit Analysis

Provides a comparison of profit generated across different regions.

The page includes:

- Regional Profit Comparison
- Segment Filter
- State Filter

Users can select a region to view detailed profit information through drill-through functionality.

---

### 8. Regional Profit Details

Displays detailed sales and profit information for the selected region.

The table includes:

- Region
- Sales
- Profit
- Category
- Sub-Category

This page is connected to the Regional Profit Analysis page using drill-through functionality.

---

### 9. Sub-Category Performance Analysis

Analyzes the performance of different product sub-categories using conditional formatting.

Key metrics include:

- Sales
- Profit
- Quantity
- Discount

Color scales, icons, and data bars are used to highlight performance differences.

---

### 10. Global Performance Dashboard

Provides an overall summary of financial and sales performance.

Key KPIs include:

- Total Sales
- Net Profit
- Total COGS
- Units Sold

The dashboard also includes:

- Sales and Profit by Country
- Monthly Sales Trend
- Profit by Discount Band
- Country Filter
